# Jenny Fairy Midday Cocktail

Landing Page for Jenny Fairy, a fashion brand. It was built using Vanilla JS, PUG, SCSS, and Vite.

- **Online version**: [[Open live demo]](https://jf-midday-cocktail.netlify.app/)
